# شات متقدم

تطبيق شات متطور مع نظام رتب وميزات متعددة.

## المميزات

### ✅ نظام المستخدمين
- تسجيل حساب جديد (اسم، كلمة سر، جنس، عمر)
- دخول الأعضاء المسجلين
- دخول كضيف (بصلاحيات محدودة)

### ✅ نظام الرتب
1. **مالك** - صلاحيات كاملة (مستخدم أساسي: محمد)
2. **وزير/وزيرة** - صلاحيات إدارية
3. **عضو مميز** - ميزات إضافية
4. **عضو** - مستخدم عادي
5. **زائر** - مستخدم ضيف

### ✅ الميزات الأساسية
- شات عام متعدد المستخدمين
- قائمة الأعضاء المتصلين
- عرض الرتب والألقاب
- نظام تفاعل (عدد الكلمات المرسلة)
- بروفايل لكل مستخدم
- محادثات خاصة

## التثبيت

1. تأكد من تثبيت Node.js
2. نزل الملفات في مجلد
3. افتح التيرمينال في المجلد

```bash
npm install
npm start
