# دردشة سوريا - منصة تعارف ودردشة

منصة دردشة وتعارف متكاملة مخصصة للمجتمع السوري، بنظام رتب متقدم ومميزات خاصة.

## المميزات الرئيسية

### نظام التسجيل
- تسجيل عضو جديد (3-14 حرف، غير مكرر)
- تسجيل الدخول للأعضاء
- الدخول كزائر (يحتفظ بحساب مؤقت)
- نظام تذكرني على الجهاز

### نظام الرتب
1. **مالك** - صلاحيات كاملة
2. **اونر** - إدارة متقدمة (لا يمكن كتم/طرد المالك)
3. **ادمن** - إدارة محدودة
4. **عضو مميز** - مميزات خاصة
5. **عضو** - صلاحيات أساسية
6. **ضيف** - صلاحيات محدودة

### مميزات البروفايل
- صورة شخصية قابلة للتغيير
- أغنية بروفايل (MP3 - 30 ثانية)
- تلوين الاسم وخلفية البروفايل
- إطار صورة متحرك
- نقاط التفاعل والذهب
- معلومات شخصية قابلة للتعديل

### نظام الذهب والمكافآت
- إرسال الذهب (المالك فقط)
- شراء الرتب بالذهب (أدمن 20K، عضو مميز 5K)
- قائمة الأثرياء (أكثر 3 أعضاء ذهباً)
- قائمة المتفاعلين (أكثر 3 أعضاء تفاعلاً)

### إدارة الدردشة
- كتم الأعضاء (5 دقائق، يوم، مؤبد)
- طرد الأعضاء
- حذف الرسائل
- نشر إعلانات على الحائط
- نظام الرد على الرسائل

### واجهة المستخدم
- وضع فاتح/داكن
- تصميم متجاوب لجميع الأجهزة
- تأثيرات دخول المستخدمين حسب الرتبة
- أزرار خاصة للوسائط (صور، فيديو، صوت، يوتيوب)
- إشارات خاصة سريعة
- قوائم قابلة للطي والإظهار
